### 1. 🏦 Kenyan Bank Loan Portfolio Dashboard
**Tools:** Power BI · MySQL · DAX

An interactive credit risk dashboard built for a fictional Kenyan commercial bank. Analyses a KES 10.3M loan portfolio across 8 branches, 20 loans, and 10 loan officers.

**Key features:**
- Live MySQL connection via Power BI Desktop
- Star schema data model — fact and dimension tables
- DAX measures: Total Portfolio, Repayment Rate, Default Rate, Fixed Default Rate (using ALL())
- Calculated column: Risk Level classification (Low/Medium/High)
- Conditional formatting: traffic light repayment rate per officer
- 2-page report: Portfolio Overview + Repayment Analysis
- Interactive slicers filtering by branch and loan type

**Insights surfaced:**
- 15% portfolio default rate — above CBK's 5% threshold
- Westlands Branch holds the largest portfolio at KES 3.2M
- Mercy Chebet (3.16%) and Peter Njoroge (9.31%) flagged as low-performing collectors

🔗 [View Live Dashboard](https://app.powerbi.com/groups/me/reports/990c595a-ee53-407b-a081-61ce4d2fa006/2cec367c823a3c99fbfc?experience=power-bi)

### 2. 👥 Kenyan HR Analytics Dashboard
**Tools:** Power BI · MySQL · DAX

A workforce analytics dashboard for Savanna Group Kenya Ltd — a fictional Kenyan company with 50 employees across 8 departments.

**Key features:**
- DAX measures: Active Employees, Attrition Rate, Average Salary
- Calculated columns: Salary Band, Years of Service
- Conditional formatting on performance scores (Red/Yellow/Green)
- Synced slicers across both report pages
- 2-page report: Workforce Overview + Performance & Leave Analysis

**Insights surfaced:**
- 8.16% attrition rate — within healthy industry range
- IT and Finance departments lead on average performance scores
- Annual leave accounts for 53% of all leave requests
- Total salary bill of KES 15.1M across 44 active employees
